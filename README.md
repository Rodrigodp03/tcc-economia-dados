# Automação e Modelagem Preditiva em Economia

Este repositório contém os códigos-fonte desenvolvidos como parte do Trabalho de Conclusão de Curso (TCC) em Ciências Econômicas.

## Objetivo
Demonstrar a transição da coleta e análise manual de dados para um ecossistema automatizado utilizando Python, Ciência de Dados e Machine Learning.

## Conteúdo do Repositório
* **Apêndice A:** Automação e visualização de indicadores (IPCA e Selic).
* **Apêndice B:** Pipeline de NLP para análise de sentimento das Atas do Copom e expectativas do Boletim Focus.
* **Apêndice C:** Modelo preditivo da taxa de câmbio utilizando Random Forest.

## Bibliotecas Utilizadas
* `pandas`, `matplotlib`, `seaborn` (Manipulação e Visualização)
* `python-bcb`, `sidrapy` (Extração de Dados via API)
* `requests`, `beautifulsoup4` (Web Scraping)
* `scikit-learn` (Machine Learning)
